# Hangman Game

## 📌 Overview

This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features

* Random word selection
* Letter-by-letter guessing
* Maximum 6 incorrect attempts
* Win/Lose conditions
* Console-based interface

## 🛠️ Technologies Used

* Python 3
* Random Module

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python hangman.py
```
